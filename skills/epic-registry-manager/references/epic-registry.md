## Usage Instructions

### Creating a New Epic
1. Check "Next Available Epic Number" above
2. Add row to the table below with your epic details
3. Increment "Next Available Epic Number"
4. Commit registry update WITH your epic file (atomic commit)

### Important Notes
- Epic numbers are globally unique across ALL features
- Never reuse epic numbers, even if an epic is deleted
- If merge conflict occurs, higher epic number wins
